Soll: Der Text auf der GDPR Seite soll sich anpassen sobald der Screen verkleinert wird.

Ist: Der Text auf der GDPR Seite passt sich automatisch an, sobald der Screen verkleinert wird.

2. Cookie banner

Soll: Der Cookie banner, soll solange auftauchen (nur wenn Javascript aktiviert ist), bis der User ihn akzeptiert. Wird er akzeptiert, so soll der Banner verschwinden.

Ist: Der Cookie banner taucht solange auf (nur wenn Javascript aktiviert ist), bis der User ihn akzeptiert. Wird er akzeptiert, so verschwindet der Banner.

3. Navbar no Javascript

Soll: Die Navbar soll ohne Javascript funktionieren.

Ist: Die Navbar funktioniert ohne Javascript.

4. Performance

Soll: Die Webseite soll mindestens einen 99% Performance Score im Lighthouse Test erreichen.

Soll: Die Webseite erreicht einen 99% bzw. auf Desktop 100% Performance Score im Lighthouse Test.
