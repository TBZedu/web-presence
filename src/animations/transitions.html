CSS Transitions

Mit CSS Transitions kann man Übergänge zwischen verschiedenen States animieren. So kann man zum Beispiel, wie Unten gezeigt, umsetzen, dass sich ein Element dreht sobald man drüber hovert. Um zu definieren was animiert werden soll gibt es das 'transition' Property. Hiermit gibt man mit, welches Property einen Übergang erhalten soll sobald sich der Wert des Properties ändert.

Beispiel mit einem Property

In diesem Beispiel wird transition: transform 2s genutzt, um dem 'transform' Property einen Übergang in der Länge von 2 Sekunden hinzuzufügen.


.animation-example__cube {
  height: 50px;
  width: 50px;
  background-color: red;
  transition: transform 2s;
}

.animation-example__cube:hover {
  transform: rotateZ(360deg);
}

Beispiel mit mehreren Properties

In diesem Beispiel wird transition: transform 1s, background-color 1.5s genutzt, um dem 'transform' Property einen Übergang in der Länge von 1 Sekunden und dem 'background-color' Property einen Übergang in der Länge von 1.5 Sekunden hinzuzufügen. Dies kann mit beliebig vielen Properties erweitert werden oder mit der Benutzung von transition: all 2s, alle Properties bestimmen. 


.animation-example__cube2 {
  height: 50px;
  width: 50px;
  background-color: red;
  transition: transform 1s, background-color 1.5s;
}

.animation-example__cube2:hover {
  transform: rotateZ(360deg);
  background-color: blue;
}
