Keyframe Animationen

Keyframes erlauben es einem Entwickler, benutzerdefinierte Animationen zu erstellen. Solch eine Animation verläuft von 0-100% während einer frei wählbaren Zeit. Man kann für jeden Fortschrittspunkt zwischen 0-100% einen Keypoint definieren, welcher den State eines bestimmten CSS Property mithilfe einer frei wählbaren Interpolationsmethode manipuliert.

Beispiel 1

Diese Animation dauert 1 Sekunde. Während dieser Zeit breitet sie sich auf 100% der Breite aus. In 30% (300ms) der Zeit macht sie jedoch 50% der Breite. Daher füllt sich die Bar zuerst schnell und wird dann langsam. Etwas ähnliches kann durch das nutzen einer anderen Interpolationsmethode (timing-function) erreicht werden. Das keyword infinite bestimmt, dass sich die Animation unendlich lang wiederholt. Um die Animation in abwechselnde Richtungen (füllen und entleeren) abzuspielen, nutzen wir das Keyword alternate. Die Farbe der Bar ändert sich während einer Sekunde, linear, von grün-gelb zu rot. 


.animation-example__cube {
  height: 50px;
  width: 50px;
  background-color: green;
  animation: myCoolAnimation1 1000ms linear 0ms infinite alternate;
}

@keyframes myCoolAnimation1 {
  0% {
    width: 0;
    background-color: greenyellow;
  }
  30% {
    width: 50%;
  }
  100% {
    width: 100%;
    background-color: red;
  }
}

Beispiel 2

Dieses Beispiel füllt die Bar zwar linear (0 Sekunden = 0% Breite, 1 Sekunde = 100% Breite), nutzt jedoch keine lineare Interpolationsmethode (timing-function). Statt linear wird hier ease-in-out verwendet. Dies sorgt dafür, dass der Anfang und das Ende der Animation leicht schneller animiert werden und gibt somit einen "smooth" Animation Effekt. 


.animation-example__cube2 {
  height: 50px;
  width: 50px;
  background-color: green;
  animation: myCoolAnimation2 1000ms ease-in-out 0ms infinite alternate;
}

@keyframes myCoolAnimation2 {
  0% {
    width: 0;
  }
  100% {
    width: 100%;
  }
}
