GIF Animationen

Am einfachsten kann man Gifs erstellen, indem man mehrere einzelne Bilder, welche sich immer nur leicht abändern, erstellt und diese mit einem Web-Tool wie beispielsweise ezgif kombiniert. Dies ergibt, wenn es schnell abgespielt wird, eine Animation so wie bei einem Daumenkino. So habe ich das auch im Beispiel umgesetzt, dass die Animation so aussieht wie ein Ball, welcher auf den Boden fällt. Die Animation wird ganz einfach mit einem Image (img) Tag eingebunden.
