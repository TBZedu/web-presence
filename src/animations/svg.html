SVG Animationen

SVG Animationen sind mit CSS relativ einfach möglich. Zuerst wird in CSS eine stroke-dasharray property hinzugefügt. Diese soll so hoch gesetzt werden, dass der ganze Pfad wieder sichtbar wird. Danach kann mit stroke-dashoffset property kontrolliert werden, wieviel vom Pfad sichtbar sein sollte. Ist der Wert 0, wird alles angezeigt. Wenn der Wert dem Wert der stroke-dasharray property entspricht, ist die hälfte sichtbar und wenn der Wert dem doppelten der stroke-dasharray property entspricht, ist wieder alles sichtbar. Dass kann so einfach in einer @keyframe rule eingebaut werden. Um mehr über @keyframes zu erfahren, kannst du hier klicken.
