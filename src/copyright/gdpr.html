DSGVO

Die Datenschutzgrundverordnung (DSGVO) / GDPR (General Data Protection Regulation) ist ein Gesetz, welches den Umgang mit personenbezogenen Daten regelt. Die DSGVO gilt seit dem 25. Mai 2018 in den Staaten der EU.

Die wichtigsten Punkte

  • Die Erhebung oder Weitergebung von personenbezogene Daten benötigt eine Zustimmung der betroffenen Person
  • Jeder hat ein Recht darauf zu erfahren welche persönlichen Daten von Unternehmen gespeichert werden
  • Jeder kann eine Löschung seiner personenbezogene Daten fordern, Unternehmen müssen diese Forderung befolgen sofern die Speicherung dieser Daten nicht durch andere gesetzliche Regelungen verpflichtet ist

Beispiele von personenbezogenen Daten

  • Name
  • Geburtsdatum / Alter
  • Anschrift
  • Telefonnummer
  • E-Mail Adresse
  • Konto- oder Kreditkartennummer
  • Personalausweisnummer
  • Kfz-Nummer
  • Krankendaten
  • Biometrische Daten
  • IP-Adresse
